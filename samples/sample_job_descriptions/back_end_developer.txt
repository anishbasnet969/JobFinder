Title: Back-End Developer (API Developer)

**Company:** Honeywell International

### About the Role
Join Honeywell International as a Back-End API Developer and play a crucial role in building the connective tissue for our innovative software solutions. You will be responsible for designing, developing, and maintaining robust APIs that facilitate seamless data exchange across our diverse systems. This position is key to enhancing our platform's scalability, security, and overall performance.

### Key Responsibilities
- Design, develop, and maintain high-performance, scalable, and secure RESTful APIs.
- Collaborate with front-end developers, product managers, and other stakeholders to define API requirements and specifications.
- Implement robust security and data protection measures, utilizing protocols like OAuth and JWT to ensure data integrity and access control.
- Create and maintain comprehensive API documentation for both internal and external developers.
- Troubleshoot and resolve API-related issues, ensuring high availability and reliability of services.
- Monitor API performance and usage to identify opportunities for optimization and improvement.

### Required Qualifications
- **Experience:** 1 to 14 years of professional experience in back-end or API development.
- **Education:** Bachelor of Business Administration (BBA) or a related field.

### Required Skills
- Proficient in API design principles and best practices.
- Strong knowledge and hands-on experience with RESTful API development.
- In-depth understanding of API security protocols, including OAuth and JWT.
- Experience with API documentation tools (e.g., Swagger/OpenAPI).
- Excellent problem-solving and analytical skills.
- Strong communication and collaboration abilities.
